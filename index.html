Ease of access

press alt + b to search in bing

press alt + g to search in google

press alt + w to search the web

press alt + i to search for images

press alt + v to search for videos

press alt + s to focus on search box

press alt + m to open settings

press alt + n to close settings

press alt + h to open help

enter "/b" in the search box to go back

enter "/ctrl" in the search box to open settings

enter "/r" in the search box to reload

press down arrow or up arrow to navigate
in autopredictions. press shift key to select the autoprediction


Open Settings

Click on settings button located on top-left corner

How to make shortcuts

right click on the shortcut buttons to edit shortcut

how to enter urls in search bar

to enter urls in search bar start with "https://" then click enter or search button. to enter file url start with "file://C:/".
